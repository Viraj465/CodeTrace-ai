What Codetrace Can Do For You

Acts as a highly knowledgeable senior engineer on your project.

travel_explore

Autonomous Research

Ask a question, and the agent proactively uses tools to search, read files, and analyze the codebase to find the exact answer.

Structural Call Graphs

Navigates class and function definitions across 6+ languages (Python, JS, TS, Java, C++, Go) to see exactly how your application is wired.

Blast Radius Analysis

Analyzes the impact of code modifications before you make them, preventing unintended breakages across your system.

Proposing Code Edits

Smart Delta Syncing

Reacts to changes in your code using SHA-256 hashing to lightning-fast re-indexes only the files you touched.

integration_instructions

IDE Context (MCP)

Connects its powerful hybrid brain directly into Cursor, Windsurf, or Claude Code for in-editor assistance instantaneously.

Privacy-First Architecture

100% Local and Air-Gapped Capable

All parsing, embedding, and graph mapping happens directly on your machine. It can be used 100% offline with zero internet connection.

  • check_circle Local LLM Required: Configure a local model via options like Ollama (llama3.2, deepseek-coder).
  • check_circle True Air-Gapped Setup: Transfer cached embedding models via USB for completely offline environments.
  • check_circle Offline Flag: Use --offline to strictly block internal telemetry and external requests.

Frictionless Setup

Get up and running in seconds. Navigate to any local codebase and type:

Terminal 
# Navigate to your project
cd /path/to/your/project

# Configure provider, download models, index code, and register MCP
codetrace init

# Launch interactive autonomous AI chat loop
codetrace chat

CLI Command Reference

codetrace init
Configures the LLM provider, downloads models, indexes the codebase, and registers MCP.
codetrace chat
Launches the interactive autonomous AI chat loop for your codebase.
codetrace index <PATH>
Forces a re-scan of a local folder or clones and indexes a GitHub URL.
codetrace config
View or update your LLM provider and API key configuration.
codetrace visualize
Generates and opens an interactive HTML graph map of your code architecture.
codetrace history
Lists all past architectural chat sessions for the current project.
codetrace export <ID>
Exports a specific chat session to the terminal or saves it as a Markdown file.

Seamless IDE Integration

Codetrace-ai acts as a Model Context Protocol (MCP) server. Running codetrace init automatically installs this configuration into Cursor and Claude Code.
